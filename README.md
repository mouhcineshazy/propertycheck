# PropertyCheck

A mobile-first application that helps renters document property conditions through photo inspections. Create timestamped records of your rental's state at move-in and move-out to protect your security deposit.

## What it does

- Take photos of your rental property organized by room
- Generate professional PDF reports with timestamps
- Compare move-in vs move-out conditions side-by-side
- Share inspection reports with landlords via secure links
